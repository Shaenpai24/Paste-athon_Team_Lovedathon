# Quick Notes
Quick Note BFS: Queue-based traversal that gives shortest path lengths in unweighted graphs.
Quick Note DFS: Depth-first traversal useful for components, cycle checks, and ordering tasks.
Quick Note Topological Sort: Orders DAG nodes so every prerequisite appears before dependent topics.
Quick Note Dijkstra: Greedy shortest path algorithm for graphs with non-negative edge weights.
Quick Note Bellman Ford: Handles negative edges and detects negative cycles with repeated relaxation.
Quick Note Union Find: Disjoint set structure for near-constant-time connectivity and merge operations.
Quick Note Segment Tree: Range query and update structure, often paired with lazy propagation.
Quick Note Fenwick Tree: Compact binary indexed tree for prefix sums and point updates.
Quick Note Knapsack: Canonical dynamic programming pattern for capacity-constrained optimization.
Quick Note LeetCode 207 Course Schedule: Checks if course prerequisite graph is acyclic.
Quick Note LeetCode 210 Course Schedule II: Returns one valid topological ordering of courses.
